# Veering Research Line

This folder collects working notes for the veering / modal exchange / mode
localization line of the CoupledBeams project.

The goal is to keep the spectral-mechanism discussion separate from the main
theory files and from the article draft. These notes are for assessment,
terminology, candidate cases, and future evidence.

## Files

- `literature_assessment.md` -- short assessment of the relevant literature.
- `terminology.md` -- working definitions and current evidence gaps.
- `candidate_cases.md` -- candidate branches and current hypotheses.
- `figures/` -- reserved for figures for this research line.
- `data/` -- reserved for derived tables or small datasets.

## Baseline Analysis Regime

- `beta = 15 deg`
- `r = 5 mm`
- `0 <= mu <= 0.9`
