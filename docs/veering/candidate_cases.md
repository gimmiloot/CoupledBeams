- Interest: control branch for comparison.

  modal order stays close to `(1,1)` over the studied range.

- Interest: shows nonmonotonic frequency behavior and an early local-order
  change on the right arm.
- Current behavior: the right arm changes from local order `1` to `2`; this
  starts before simple proximity to the higher right-arm reference family is
  enough to explain the branch.
- Working hypothesis: possible exchange of modal character.

## `bending_desc_04`

- Interest: strongest current candidate because it shows a staged
  reorganization of local modal order.
- Current behavior: the branch changes from `(2,2)` through an intermediate
  pattern toward `(1,3)`, with a nonmonotonic frequency curve.
- Working hypothesis: strongest candidate for veering / modal exchange.
