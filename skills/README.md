# Project skills

This folder contains project-specific writing and workflow skills.

Use these files when working on text intended for readers outside the project,
especially journal papers, abstracts, cover letters, and reviewer responses.

## Current skills

- `external_writing_general.md`
  General rules for any external-facing text.

- `paper_dorofeev_style_article_writing.md`
  Article-specific writing skill for the current CoupledBeams paper in
  `paper_dorofeev_style/`.

## Policy

- Keep `AGENTS.md` short.
- Put long, reusable writing guidance here.
- Add new skills as separate files rather than overloading one file.
