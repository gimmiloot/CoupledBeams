# AGENTS.md

## Цель проекта
Этот репозиторий предназначен для второй научной задачи по исследованию связанных балок и связанных с ними аналитических и численных моделей.

## Source Of Truth
При работе с проектом ориентируйся в первую очередь на:
- `docs/theory/`
- `docs/literature/`
- `docs/project_log/journal.md`

Если между кодом и этими материалами есть расхождение, не меняй смысл модели молча: сначала согласуй изменения с теоретическими материалами.

## Code Layout
- `src/my_project/analytic/` — аналитические модули и выводы, реализованные в коде.
- `src/my_project/fem/` — FEM-часть и связанные численные сборки.
- `scripts/` — отдельные запускаемые сценарии для воспроизводимых задач.

## Правила работы
- Не меняй обозначения, если это не согласовано с `docs/theory/`.
- При появлении новых допущений обновляй `docs/theory/assumptions.md`.
- Вычислительные результаты сохраняй в `results/`, а не в `src/`.
- Теоретические заметки, derivations и допущения держи в `docs/theory/`.
- Заметки по литературе и индексацию источников держи в `docs/literature/`.

## Минимальный рабочий принцип
Предпочитай маленькие прозрачные изменения, чтобы структура репозитория оставалась пригодной для дальнейшего научного развития и воспроизводимых запусков.
